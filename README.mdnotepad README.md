# MeteoGalle 🌤

App meteorológica en tiempo real desarrollada con Python y Flask.

## Demo

🔗 https://appmeteo-h0yg.onrender.com

## Descripción

Ingresás el nombre de cualquier ciudad del mundo y obtenés al instante
la temperatura, sensación térmica, humedad y velocidad del viento.
Diseño minimalista oscuro con detalles en dorado.

## Tecnologías

- Python
- Flask
- API OpenWeatherMap
- HTML / CSS
- Render (deploy)

## Funcionalidades

- Consulta del clima en tiempo real para cualquier ciudad del mundo
- Temperatura en Celsius
- Sensación térmica, humedad y viento
- Ícono del estado del cielo
- Diseño responsive para celular y escritorio
- Deployada y accesible desde cualquier dispositivo
